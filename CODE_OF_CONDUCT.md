# Code of Conduct

Ce code s’applique aux issues, PR, discussions et échanges liés au projet.

- Respect : pas de propos discriminants, agressifs ou hostiles.
- Collaboratif : clarifier, documenter, aider à reproduire les problèmes.
- Feedback : critique factuelle et orientée solution.
- Sécurité/vie privée : pas de secrets ni données privées dans les issues/PR.
- Signalement : en cas d’écart, mentionner le problème dans l’issue/PR concernée ou ouvrir une issue dédiée.
